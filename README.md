# Парсер номеров Avito для компании Директ-Лайн

## 1. Подготовка

1. Положи Excel-файл со ссылками на объявления в корень проекта (рядом со `smart_pars_avito.py`).  

2. При необходимости очисти старые результаты:
   - `avito_phones_playwright/phones_map.json` — оставь в файле просто `{}`  
   - `phones.xlsx`  удали старые строки с номерами и ссылками.


## 2. Настройка скрипта
1. В файле `smart_pars_avito` в поле `INPUT_FILE` замени название файла на вай файл из пункта 1

2. В файле `smart_pars_avito` в поле `TEST_TOTAL` укажите нужное сколько номеров хотите спарсить

3. По надобности введите данные для проксирования.

4. Настрой `CONCURRENCY`  # сколько вкладок одновременно (2–3 оптимально)

PS.Остальное лучше не меняй)

## 3. Запуск скрипта
1. В файле `smart_pars_avito` нажми на зеленый треуголинк справа сверху
2. Дождись открытия бразуера, залогинься в аккаунт авито и наэми ENTER в консоли (продублировано будет в консоли)
3. Дождись завершения работы скрипта

## 4. Создание EXCEL

1. После того как отработал скрипт парсинга переди в файл `make_excel_from_ocr_output` и нажми на зеленый треуголинк справа сверху
2. Дождись завершения работы программы и получай готовый результат в файле `phones.xlsx`
