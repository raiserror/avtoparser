# Парсер номеров Avito для компании Директ-Лайн

## 1. Подготовка

1. Положите Excel-файл со ссылками на объявления в корень проекта (рядом со `smart_pars_avto.py`).  

2. При необходимости очистите старые результаты:
   - `avito_phones_playwright/phones_map.json` — оставьте в файле просто `{}`  
   - `phones.xlsx`  удалите старые строки с номерами и ссылками.


## 2. Настройка скрипта
1. В файле `smart_pars_avto` в поле `INPUT_FILE` замените название файла на файл из пункта 1

2. В файле `smart_pars_avto` в поле `TEST_TOTAL` укажите нужное количество номеров, которое хотите спарсить

3. По надобности введите данные для проксирования.

4. Настрой `CONCURRENCY`  # сколько вкладок одновременно (2–3 оптимально)

PS.Остальное лучше не меняйте

## 3. Запуск скрипта
1. В файле `smart_pars_avto` нажмите на зеленый треуголинк справа сверху
2. Дождись открытия бразуера, залогинься в аккаунт авито и нажмите ENTER в консоли (продублировано будет в консоли)
3. Дождитесь завершения работы скрипта

## 4. Создание EXCEL

1. После того как отработал скрипт парсинга перейдите в файл `make_excel_from_ocr_output` и нажмите на зеленый треуголинк справа сверху
2. Дождитесь завершения работы программы и получайте готовый результат в файле `phones.xlsx`
